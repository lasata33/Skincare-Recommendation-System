The Skincare Recommendation System is a web application that helps users identify their skin type and receive personalized skincare suggestions. 
It includes a quiz to determine skin type, a recommendation engine that suggests products, an ingredient analyzer to check product safety, 
and a water tracker to monitor hydration. An admin dashboard is also provided for managing users, products, and quizzes.

This project is built using PHP for the backend, MySQL for database management, and HTML, CSS, Bootstrap, and JavaScript for the frontend. 
Development and testing were carried out using XAMPP and phpMyAdmin, with GitHub for version control.

To run the project locally, clone the repository, move the folder into the htdocs directory of XAMPP, start Apache and MySQL, 
and import the provided SQL file into phpMyAdmin. Once the database is set up, open your browser and navigate to http://localhost/skincare-recommendation/index.php.

Screenshots of the user interface, including the quiz, recommendation page, ingredient analyzer, water tracker, and admin dashboard, are available in the 
/screenshots folder to give a quick overview of the system.

Future improvements may include adding product reviews, expanding the recommendation logic, and developing a mobile app version to make the system more accessible.
