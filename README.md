# 🌸 Skincare Recommendation System

A simple yet powerful **web application** that helps users identify their skin type and receive personalized skincare suggestions. The system combines quizzes, recommendations, ingredient analysis, hydration tracking, and an admin dashboard in one clean interface.

---

## 🚀 Features
- Secure user registration and login  
- Skin type quiz with personalized recommendations  
- Ingredient analyzer for product safety checks  
- Water tracker to monitor hydration intake  
- Admin dashboard to manage users, products, and quizzes  

---

## 🛠️ Tech Stack
- **Frontend:** HTML, CSS, Bootstrap, JavaScript  
- **Backend:** PHP  
- **Database:** MySQL (via phpMyAdmin)  
- **Environment:** XAMPP (Apache + MySQL)  

---

## 📂 Project Structure
- `index.php` → Homepage and navigation  
- `quiz.php` → Skin type quiz module  
- `recommendation.php` → Personalized product suggestions  
- `ingredient.php` → Ingredient analyzer  
- `water.php` → Hydration tracker  
- `admin/` → Admin dashboard (manage users, products, quizzes)  
- `db.php` → Database connection  
- `README.md` → Project documentation
